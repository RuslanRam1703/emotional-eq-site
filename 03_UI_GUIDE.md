1\. Назначение UI-системы



UI Emotional EQ строится как минималистичная, кинематографичная интерфейсная система, основанная на эмоциональном восприятии состояния пользователя.



Основная задача UI — не визуальная выразительность, а поддержка эмоционального выбора без перегрузки интерфейса.



2\. Визуальная концепция



UI основан на сочетании:



тёмного фона

мягкого стеклянного слоя (glassmorphism)

цветовых эмоциональных акцентов

кинематографического освещения



Интерфейс должен восприниматься как спокойная цифровая среда.



3\. Цветовая система

3.1 Базовые цвета

Background: почти чёрный / глубокий графит

Surface: полупрозрачные тёмные слои

Text Primary: светлый нейтральный

Text Secondary: приглушённый серый

3.2 Эмоциональные акценты



Цвет зависит от состояния пользователя:



спокойствие → холодный синий

энергия → тёплый оранжевый

тревожность → мягкий фиолетовый

радость → жёлто-золотой

меланхолия → приглушённый сине-серый

3.3 Принцип цвета



Цвет НЕ декоративный, а функциональный:



отражает состояние пользователя

влияет на glow / подсветку UI

используется минимально

4\. Типографика

Принципы:

современный sans-serif

высокая читаемость

отсутствие декоративных шрифтов

умеренный контраст

Иерархия:

H1: крупный, эмоциональный заголовок

H2: секции интерфейса

Body: основной текст

Caption: вспомогательные пояснения

5\. Пространство и сетка

Принципы:

много воздуха

крупные отступы

центрированная композиция

отсутствие плотных UI-блоков

Сетка:

desktop-first layout

адаптивная сетка (responsive)

минимум 12-column логика (не жёстко)

6\. Компоненты интерфейса

6.1 Button

мягкие скругления

лёгкий blur background

hover: усиление glow

без резких контрастов

6.2 Card (Movie Card)

стеклянная поверхность

мягкая тень

акцент на постере

минимум текстовой перегрузки

6.3 Slider (Emotional EQ)

5 независимых контролов

плавная анимация

визуальная ассоциация с состоянием

отсутствие “технического” вида

6.4 Preset Tile

крупные интерактивные блоки

цветовое кодирование эмоции

hover: лёгкое увеличение + glow

7\. Glassmorphism правила

Разрешено:

blur background

semi-transparent layers

мягкие градиенты

Запрещено:

агрессивные неоновые эффекты

перегруженные стеклянные слои

“киберпанк” визуал

8\. Анимации

Принципы:

медленные и плавные

без резких переходов

естественные easing curves

микроинтеракции, а не эффектность

9\. Состояния UI

Hover

усиление свечения

лёгкое масштабирование

Active

мягкое “вдавливание”

усиление цвета

Loading

минимальный motion

отсутствие перегрузки визуалом

10\. Ограничения UI

запрещены декоративные элементы без функции

нельзя менять эмоциональную цветовую систему

нельзя вводить новые визуальные стили без документации

нельзя отходить от glass + dark + cinematic системы

11\. Итог UI-системы



UI Emotional EQ — это:



минималистичный, эмоционально-реактивный интерфейс, который визуально отражает состояние пользователя и поддерживает процесс выбора контента без когнитивной перегрузки
